Feature Correlation Analysis

This visualization shows the correlation matrix between different features in the streaming platform dataset. The color intensity represents the strength and direction of correlation between pairs of features.

Key Insights:

  • Strong positive correlation (0.82) between content type and duration unit, indicating a relationship between content type (movie/TV show) and how duration is measured.
  • Strong negative correlation (-0.82) between type and duration_numeric, showing that different content types have distinct duration patterns.
  • Strong correlation (0.94) between genre_list and listed_in features, suggesting redundancy in how genres are represented.
  • Significant negative correlation (-0.7) between date_added and platform, indicating different platforms have distinct content addition patterns.

Feature Importance for Platform Prediction

This bar chart shows the relative importance of different features in predicting which streaming platform a title belongs to. It uses a Random Forest model to calculate feature importance scores.

Key Insights:

  • Date added is by far the most important feature for determining platform, suggesting each platform has distinct content acquisition timelines.
  • Country of origin is the second most important feature, showing that platforms differentiate in their international content strategy.
  • Show ID, rating, and genre categories also contribute meaningfully to platform differentiation.
  • Content type (movie vs. TV show) is surprisingly one of the least important features, despite the clear differences in content type distribution shown in earlier analyses.

PCA Explained Variance

This plot shows the Principal Component Analysis (PCA) of the dataset features, displaying both individual and cumulative explained variance for each principal component.

Key Insights:

  • The first principal component explains approximately 25% of the variance in the data.
  • The first 5 components together explain approximately 65-70% of the total variance.
  • It takes about 12 principal components to reach 90% explained variance, indicating the complexity of the dataset.
  • The gradual slope of the cumulative variance line suggests no clear "elbow point," meaning there isn't an obvious dimensionality reduction without losing information.

Selected Features Correlation

This visualization shows the correlation matrix for features specifically selected for the recommendation system. It highlights relationships between features that were deemed most relevant for content recommendations.

Key Insights:

  • Strong correlations exist between genre-related features, indicating that certain genres tend to co-occur.
  • Recency score shows strong correlations with both release year and decade, as expected.
  • Content age and release year have perfect negative correlation.
  • The popularity proxy shows strong correlation with cast count, suggesting titles with larger casts may be more popular.
  • TF-IDF PCA components show correlation patterns with certain genres, indicating text-based content descriptors have genre associations.

Feature Selection Results

This visualization shows all features that were selected for inclusion in the final recommendation system model.

Key Insights:

  • A diverse set of features were selected, spanning content metadata, genre information, text analysis features, and engagement metrics.
  • Genre-specific features (Drama, Comedy, Action, etc.) were all included, highlighting the importance of genre in content recommendations.
  • Platform-specific genre interactions (Netflix Drama, Amazon Prime Comedy, etc.) were also included to capture platform-specific preferences.
  • Text-based features (TFIDF PCA components) were included to capture semantic similarity between content descriptions.
  • Both temporal features (release_year, content_age) and audience features (min_age, rating) were deemed important for recommendations.

Feature Selection Frequency

This bar chart shows how frequently each feature was selected across different feature selection methods, highlighting the most consistently important features.

Key Insights:

  • The top 9 features were selected by all feature selection methods, indicating strong consensus on their importance.
  • Content-related features like International Movies on Netflix, actor/director information, cast count, and content age were universally selected.
  • Engineered features like movie_duration, popularity_proxy, and recency_score were also consistently selected, validating their creation.
  • Platform-specific genre features (like Netflix Drama or Hulu Comedy) were selected less frequently, suggesting they may be useful but not essential.

Random Forest Feature Importance

This bar chart shows the feature importance scores from a Random Forest model, highlighting which features the model found most useful for making predictions.

Key Insights:

  • Platform encoding is by far the most important feature, suggesting that the streaming platform itself is a strong predictor of content characteristics.
  • Description length is the second most important feature, indicating that the amount of detail in content descriptions correlates with other content attributes.
  • Actor and cast information are also important, suggesting that the talent involved in content production provides valuable signals.
  • Popularity proxy, an engineered feature combining multiple signals, ranks among the top 5 features, validating its usefulness.

Mutual Information Feature Scores

This visualization shows feature importance as measured by mutual information, which captures non-linear relationships between features and the target variable.

Key Insights:

  • Temporal features dominate mutual information scores, with release_year, recency_score, and content_age having the highest values.
  • Decade is also important, further emphasizing that time-based features are crucial for content prediction.
  • Text-based features (TFIDF PCA components) have moderate mutual information scores, indicating they capture useful but not dominant signals.
  • Description length and duration features show lower but still significant mutual information scores.
