Conclusions

Model Performance Summary

Key Findings:

  • XGBoost is the best performing model with 99% accuracy, precision, recall, and F1 score.
  • Random Forest is a close second with 94% accuracy and strong performance across all classes.
  • Both ensemble methods (XGBoost and Random Forest) significantly outperform Logistic Regression.
  • The most important features across all models are cast_count, actor information, and genre_count.
  • The recommendation system would benefit most from deploying the XGBoost model in production.

Implementation Recommendations:

  • Deploy XGBoost as the primary recommendation engine
  • Focus on ensuring accurate cast and genre data as these are crucial predictive features
  • Consider using Random Forest as a backup model due to its strong performance
  • Further optimize the XGBoost model with hyperparameter tuning focused on classes with lower performance
  • Implement regular model retraining as new content is added to ensure continued accuracy
